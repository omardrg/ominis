== Description ==
Ominis is a clean and flexible WordPress theme designed for blogs, portfolios, and business websites.  
It includes Bootstrap 5 components, a responsive navigation menu, customizable theme options, and support for galleries and masonry layouts.

== Installation ==
1. Upload the theme to `/wp-content/themes/`.
2. Activate it from Appearance → Themes.
3. Assign a menu to the “Primary Menu” location.
4. Configure theme options under Appearance → Customize.
